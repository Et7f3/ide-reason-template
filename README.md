# How to install

clone this repository

```
cd josselin
npm install -g esy@latest bs-platform
esy install
bsb -init .
```


# How to build (try to build after the install)
```
esy build (for native)
npm run bs:build (for js)
```


# How to launch utop
```
esy utop
```

# How to launch utop with loaded module
```
esy butop
```
